# DESIGN.md

## 气质与意象
专业的企业级 SaaS 产品，为 AI Agent 提供可靠的预约基础设施。视觉风格应体现：精准、可信赖、高效。

关键词：日程网格、时间线、清晰的层级、数据驱动。

## 视觉策略
- 图形语言：克制的几何图形、清晰的数据表格、时间线视图
- 图标风格：线性图标（Lucide），统一 stroke-width
- 卡片设计：轻微阴影、清晰边框，避免过度装饰

## 配色方案
- 主色调：neutral（中性灰）- 专业、冷静、技术感
- 强调色：用于操作按钮和关键状态
- 状态色：成功(green)、警告(amber)、错误(red)、信息(blue)

## 字体排版
- 字体族：Business（商务风格）
- 标题：font-semibold，层级清晰
- 正文：font-normal，易于阅读
- 数据/代码：等宽字体用于 API Key、时间戳等

## 动效与交互
- 过渡：150-200ms 的快速响应
- 状态变化：淡入淡出，避免突兀
- 表格操作：行内编辑优先，减少页面跳转

## 页面结构
- 左侧导航：固定侧边栏，主要功能入口
- 内容区：面包屑 + 标题 + 操作区 + 数据展示
- 响应式：移动端收起侧边栏为抽屉

## 设计禁忌
- 禁止使用蓝紫色渐变（AI 味太重）
- 禁止过度装饰性动画
- 禁止模糊不清的状态表达
- 禁止隐藏关键操作入口
